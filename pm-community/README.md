# PM社区 - 产品经理交流平台

## 📖 项目简介

PM社区是一个专为产品经理打造的交流平台，提供产品分类讨论、闲聊区、工具箱、创作中心等功能，帮助产品经理交流经验、分享知识、共同成长。

## 🎯 核心功能

### 1. 首页内容流
- 推荐/最新/热门/关注内容切换
- 文章卡片展示（标题、摘要、分类、点赞、评论）
- 问答卡片展示（问题状态、回答数）
- 快捷功能导航
- 热门话题标签

### 2. 分类广场
- 6大产品分类：C端产品、B端产品、AI/大数据、用户增长、策略产品、产品运营
- 文章/问答/讨论类型筛选
- 排序功能（最新、最热）
- 分类导航

### 3. PM茶水间（闲聊区）
- 8个话题频道（摸鱼时间、求职交流、求助问答等）
- 实时消息发送
- 在线用户展示
- 快捷表情选择
- 本周活跃榜

### 4. 工具箱
- 需求模板库
- PRD助手（AI）
- 原型托管
- 数据看板
- 竞品分析
- 流程图工具
- 思维导图
- 需求池管理
- 热门模板下载

### 5. 创作中心
- 三种创作类型：文章、问答、讨论
- 富文本编辑器
- 分类和标签管理
- 封面图上传
- 可见性设置（公开/粉丝可见/仅自己）

### 6. 内容详情页
- 正文展示
- 文章目录导航
- 点赞、收藏、分享
- 评论系统
- 相关推荐
- 作者信息卡

### 7. 个人主页
- 用户信息展示
- 数据统计（获赞、粉丝、关注、文章）
- 6个标签页：文章、问答、讨论、收藏、粉丝、关注
- 个人设置入口

### 8. 消息通知
- 点赞、评论、关注、系统消息分类
- 未读标识
- 消息统计
- 通知设置

### 9. 账号系统
- 注册流程（分步式）
  - 步骤1：基本信息（昵称、性别、邮箱、手机、密码）
  - 步骤2：完善资料（职业身份、公司行业、工作年限、兴趣领域）
- 登录方式：密码登录、验证码登录
- 第三方登录：GitHub、微信、钉钉
- 账号安全：修改密码、绑定手机、绑定邮箱
- 隐私设置：粉丝可见、收藏可见、在线状态、私信设置
- 通知设置：邮件通知、网站通知

## 🎨 设计特色

### 色彩系统
- **主色调**: 紫色渐变 (#6366f1 → #4f46e5)
- **成功**: 绿色 (#10b981)
- **警告**: 橙色 (#f59e0b)
- **危险**: 红色 (#ef4444)
- **背景**: 白色 (#ffffff) / 浅灰 (#f8fafc)

### 交互效果
- 卡片悬浮动效
- 按钮点击反馈
- 页面切换动画
- Toast提示动画
- 模态框滑入效果

### 响应式布局
- 桌面端：三栏布局（侧栏 + 内容 + 侧栏）
- 平板端：双栏布局
- 移动端：单栏布局

## 🚀 使用说明

### 本地运行
1. 直接在浏览器中打开 `index.html`
2. 或使用本地服务器：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx serve
   ```

### 功能演示
- **点赞/收藏**: 点击卡片上的点赞或收藏按钮
- **关注用户**: 点击用户卡片或文章作者旁的关注按钮
- **切换Tab**: 点击顶部导航或内容区的Tab标签
- **发送消息**: 在闲聊区输入框输入内容后按Enter或点击发送
- **登录注册**: 点击右上角的登录/注册按钮

### 快捷键
- `ESC`: 关闭弹窗
- `Ctrl/Cmd + K`: 聚焦搜索框
- `Ctrl/Cmd + N`: 新建内容

### 控制台命令
在浏览器控制台可以输入：
- `login()`: 模拟登录状态
- `logout()`: 退出登录
