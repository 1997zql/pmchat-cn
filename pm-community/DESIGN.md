## 📋 项目概述

**项目名称**: PM社区
**项目类型**: Web端产品经理垂直社区平台

## 🎨 设计规范

### 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#6366f1` | Indigo，主品牌色 |
| 主色深 | `#4f46e5` | 按钮悬停、强调 |
| 主色浅 | `#818cf8` | 次要元素 |
| 辅助色 | `#f0abfc` | 渐变搭配 |
| 强调色 | `#f472b6` | 特殊标记 |
| 成功色 | `#10b981` | 成功状态 |
| 警告色 | `#f59e0b` | 警告提示 |
| 危险色 | `#ef4444` | 错误、删除 |
| 背景色 | `#ffffff` | 主背景 |
| 次背景 | `#f8fafc` | 卡片背景 |
| 文字主色 | `#1e293b` | 标题、重要文字 |
| 文字次色 | `#64748b` | 正文内容 |
| 文字弱色 | `#94a3b8` | 辅助信息 |
| 边框色 | `#e2e8f0` | 分割线、边框 |

### 分类主题色

| 分类 | 渐变色 | 图标 |
|------|--------|------|
| C端产品 | `#667eea` → `#764ba2` | 手机图标 |
| B端产品 | `#f093fb` → `#f5576c` | 建筑图标 |
| AI/大数据 | `#4facfe` → `#00f2fe` | 大脑图标 |
| 用户增长 | `#43e97b` → `#38f9d7` | 图表图标 |
| 策略产品 | `#fa709a` → `#fee140` | 架构图标 |
| 产品运营 | `#a8edea` → `#fed6e3` | 钻石图标 |

### 字体规范

- **主字体**: Inter, -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif
- **标题字重**: 600-700
- **正文字重**: 400-500
- **基础字号**: 14px
- **行高**: 1.6

### 圆角系统

| 元素 | 圆角值 |
|------|--------|
| 小按钮/标签 | 6px |
| 中等元素 | 10px |
| 大卡片/容器 | 16px |
| 圆形头像 | 50% |
| 胶囊按钮 | 9999px |

### 阴影系统

- **微阴影**: `0 1px 2px rgba(0, 0, 0, 0.05)`
- **中阴影**: `0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)`
- **大阴影**: `0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05)`

## 📐 布局结构

### 页面布局 (1400px 最大宽度)

```
┌─────────────────────────────────────────────────────────────────┐
│                           Header (64px)                         │
├────────────┬────────────────────────────────┬───────────────────┤
│            │                                │                   │
│  左侧边栏   │         主内容区域             │    右侧边栏        │
│  (260px)   │        (自适应)                │    (320px)        │
│            │                                │                   │
│ - 功能导航  │ - Tab切换 (推荐/最新/热门/关注)  │ - 登录注册卡片     │
│ - 热门话题  │ - 内容卡片列表                 │ - PM茶水间(闲聊)   │
│ - 社区数据  │ - 分类网格                     │ - 推荐关注         │
│            │                                │                   │
└────────────┴────────────────────────────────┴───────────────────┘
```

### 响应式断点

| 屏幕宽度 | 布局变化 |
|----------|----------|
| > 1200px | 三栏布局 |
| 992-1200px | 左右栏缩小 |
| < 992px | 单栏布局，侧栏隐藏 |
| < 768px | 移动端适配 |

## 🔧 功能模块

### 1. 顶部导航栏

- **Logo区域**: 品牌标识 + 产品名称
- **主导航**: 首页、分类、闲聊、工具箱
- **搜索框**: 全局搜索 (280px宽度)
- **发布按钮**: 主色调 CTA
- **用户菜单**: 头像 + 通知铃铛

### 2. 左侧功能导航

#### 常用功能 (快捷入口)
- ✏️ 写文章
- ❓ 提问
- 📄 需求模板
- 📊 数据看板
- 📚 学习中心
- 👥 求职招聘

#### 热门话题
- 横向标签云，点击跳转话题页

#### 社区数据
- 在线PM数量
- 今日帖子数
- 新加入用户数

### 3. 主内容区

#### Tab切换
- 推荐 (默认)
- 最新
- 热门
- 关注

#### 内容卡片
- 用户头像 + 昵称 + 身份
- 分类标签
- 文章标题
- 摘要预览
- 互动数据 (点赞/评论/收藏)
- 话题标签

#### 产品分类网格 (6个)
- C端产品
- B端产品
- AI/大数据
- 用户增长
- 策略产品
- 产品运营

### 4. 右侧边栏

#### 登录注册卡片
- 主色调渐变背景
- 注册/登录按钮
- 第三方登录 (GitHub/微信/钉钉)

#### PM茶水间 (闲聊区)
- 实时消息列表
- 输入框 + 发送按钮
- 轻量化社交互动

#### 推荐关注
- KOL用户推荐
- 关注按钮

### 5. 登录弹窗

- 邮箱/手机号登录
- 记住我 + 忘记密码
- 第三方登录
- 注册引导

## 🎭 交互规范

### 按钮交互

| 状态 | 效果 |
|------|------|
| 默认 | 基础样式 |
| 悬停 | 上移1px + 阴影加深 |
| 点击 | 按下反馈 |
| 禁用 | 降低透明度 |

### 卡片交互

| 状态 | 效果 |
|------|------|
| 默认 | 微阴影 |
| 悬停 | 上移2-4px + 阴影加深 |

### 收藏功能

- 点击切换实心/空心图标
- 图标颜色变化

### 关注功能

- 点击切换"关注"/"已关注"状态
- 颜色状态联动

## 📱 设计亮点

1. **渐变色块**: 分类图标使用渐变色，视觉层次丰富
2. **卡片悬浮**: 微妙的位移和阴影变化，提升层次感
3. **圆角系统**: 统一的圆角规则，一致性强
4. **色彩心理学**: 紫色系传达专业、创新感
5. **响应式布局**: 适配多端设备
6. **登录模态框**: 渐进式披露，不打断浏览流程
